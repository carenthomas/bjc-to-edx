What kind of pattern was alpha?

In the big picture, what kind of pattern was that entire MapReduce operation (alpha)?

Mapping
No; mapping patters apply a transformation to every element in a set.
Finding
Right!
Counting
No; in counting algorithms, a number is returned that relates to some quantity in the set.
Filtering
Only sort of; in filtering patterns a part of a set is selected and returned. There is a more appropriate pattern for the alpha MapReduce.
Testing
No.
Combining
No; in combining patterns a set of things is combined into a single thing.
